# SIP
 Sistema Previsional Inteligente

# Pasos para instalar proyecto
```
 1- composer install
 2- php artisan key:generate
 3- php artisan cache:clear
 4- php artisan config:cache
 6- npm run production
 7- php artisan serve
```
